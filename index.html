DÚVIDAS?

QR Code é um código de barras bidimensional, que pode ser facilmente escaneado usando a maioria dos telefones celulares equipados com câmera.

É uma ferramenta utilizada para gerar diferentes tipos de QR Code. Você pode criar seu QR Code para abrir um site, uma página de rede social ou até um vídeo.

Basta inserir no campo de texto a URL do site ou página que deseja criar um QR Code, selecione também o formato do arquivo e tamanho da imagem, clique em Criar e seu QR Code já estará disponível, para baixar o QR Code clique no botão de Download.

Sim, você pode criar infinitos QR Codes de forma totalmente gratuita.

Se a URL não for alterada ou excluída os QR Codes permanecem ativos por quanto tempo quiser.
